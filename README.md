# Auction-Webapp-Model
A model for an auction web application implemented with easyui, jsp, jquery, ajax, etc. Project for learning web app development.

Start time: 2015-6-13. 

Status: Finished (for now).

Authors: Chu Zhang and Yue Zheng.

Purpose: Build a model for an auction web application to learn about CRUD application in web application development using mainly easyui. Features implemented: placing bids(with validity checks), adding books for auction, editing book information, checking bid histories, automatically ending auctions when idle for over two hours, automatic reloads, automatic alerts when auction ends.

Roles: Frontend--Yue Zheng, backend--Chu Zhang
